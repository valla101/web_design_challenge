Weather Data & Visualization

With easier access to data with the emergence of API's, it has now become easier to gather and visualize datasets. This dataset in particular was generated using Python to query the OpenWeatherMap API. Cities would be generated randomly and the location of these cities were used to pull weather data (Including Temperature, Wind, Humidity, Current Forecast, and more)


This project was made possible using Weather Data from the OpenWeatherMap API. Check Them Out!
